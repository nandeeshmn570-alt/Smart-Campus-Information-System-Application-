# Smart Campus Information System Application

## Introduction

The Smart Campus Information System Application is a Python-based project developed to manage various academic and administrative activities within a college campus. The project includes modules such as student registration, course enrollment, student record management, searching and sorting, file handling, directory scanning, fee calculation, and performance analytics.

# How to Run

## 1. Install Required Libraries

Open terminal or command prompt and run:

```bash id="7dnnif"
pip install numpy pandas matplotlib
```

---

## 2. Open the Project Folder

Open the project folder in:

* VS Code
* PyCharm
* Terminal / CMD

---

## 3. Run the Program

Execute the following command:

```bash id="z5k20m"
python main_application.py
```

If Python is not recognized, use:

```bash id="n6s6b5"
py main_application.py
```

---

## 4. Use the Menu

After running the program, the Smart Campus menu will appear. Enter the required option number to access a module.
